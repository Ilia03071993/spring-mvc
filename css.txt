Атрибуты - используются в tags, требуются для css или js, служат как метки для использования, называют их селекторами (по id, по class), id - может быть только один на странице, а class может быть одинаковый.


h1-h6 - заголовки

p - обзац, пораграфю
span - пораграф отличается от p тем, что не переносится на новую строчку (in line text).

input - служит для ввода данных, имеет атрибуты:
type=("text" or "password"), 
value="login" - то что уходит на сервер,
placeholder - отображение текста на фоне.

br - перенос строки.


Сss подключается в link href="../css/style.css".

Для селекторов используются обозначения:
id - # ,
class - . ,
tag - name tag.
