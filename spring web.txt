    1. Что такое Клиент-Серверная архитектура?
Клиент-серверная архитектура – шаблон проектирования, основа для создания веб-приложений. Данная архитектура состоит из трех компонентов: клиент, сервер, сеть.

    2. Что такое IP-адрес?
IP адресс - глобальный адрес, использующийся для уникальной идентификации компьютеров в составной сети.

    3. Что такое localhost?
Localhost – стандартное, официально зарезервированное название для частных IP-адресов. По смыслу – это IP-адрес нашего компьютера.

    4. Что такое порт?
Порт – число от 1 до 65535.  
Числовой идентификатор программы или процесса, которые обслуживают сетевые соединения на заданном IP-адресе. Номера портов не повторяются.

    5. Что нужно знать для отправки запроса на сервер?
Для обращения к серверу необходимо знать его IP и порт.

    6. Как определяется IP-адрес любого сайта, если изначально он неизвестен?
Определяется IP-адрес любого сайта можно с помощью DNS протокола.
Если через Windows узнать IP адресс, можно через cmd, ip адресс текущий -> ipconfig, внешний -> ping google.com.
DNS – протокол (служба), определяющая IP-адрес на основании доменного имени.
Пример: в URL браузера введено google.com. Служба DNS преобразует доменное имя в IP и реальный запрос будет вида: 
8.8.8.8 (ip-адрес google).

    7. Что такое протокол?
Протокол - это набор правил и форматов, которые определяют способы взаимодействия между клиентом и сервером веб-приложения.

    8. Что такое HTTP?
HTTP(Hypertext Transfer Protocol) – протокол передачи гипертекста или протокол передачи данных по сети.

    9. В чем суть разделения методов HTTP на GET, POST и т.д.?
Методы HTTP (Hypertext Transfer Protocol) определяют, как клиент и сервер взаимодействуют друг с другом при передаче данных. Они указывают, какие действия должен выполнить сервер при получении запроса от клиента.

GET  - используется для запроса данных от сервера. Клиент отправляет запрос на получение определенной информации, и сервер возвращает эту информацию в ответ.

POST - используется для отправки данных на сервер для обработки. Например, когда пользователь заполняет форму на веб-сайте, данные из этой формы отправляются на сервер с помощью метода POST для обработки.

PUT - используется для загрузки данных на сервер. Клиент отправляет данные на сервер для сохранения или обновления.

DELETE - используется для удаления данных на сервере. Клиент отправляет запрос на удаление определенных данных, и сервер удаляет эти данные

10.  Какой единственный HTTP-метод можно использовать в адресной строке браузера?
GET - единственный HTTP-метод, который можно использовать в адресной строке браузера. Когда вы вводите URL в адресной строке браузера и нажимаете Enter, браузер отправляет GET-запрос на сервер для получения запрошенной страницы или данных.
